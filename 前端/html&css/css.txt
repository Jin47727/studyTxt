1.概念:
    css(Cascading Style Sheets) 美化样式:主要用于设置html页面中的文本内容(字体,大小,对齐方式等),图片的外形(宽高,边框样式,边距)以及版面的布局等外观样式.
 2.引入样式表(位置不同而已):
    1.内部样式表
         在页面内的任何位置都可,一般是<head></head>里面
         <style>
            选择器 { 属性 : 值;}
         </style>
    2.行内式(内联式)
        <标签名 style="属性1:属性值1;属性2:属性值2;"></标签名>
    3.外部样式表(外联式)
        在页面内的任何位置都可,一般是<head></head>里面
        <link rel="stylesheet" href="css/style.css" />
3.标签选择器
    th td  :直接用标签的名字
    #id :id选择器
    .class  : 类选择器
